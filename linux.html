Linux Debian 5

Debian adalah sistem operasi bebas yang dikembangkan secara terbuka oleh banyak programer sukarela(pengembang Debian) yang tergabung dalam Proyek Debian. Sistem operasi Debian adalah gabungan dari perangkat lunak yang dikembangkan dengan lisensi GNU, dan utamanya menggunakan kernel Linux, sehingga populer dengan nama Debian GNU/Linux. Sistem operasi Debian yang menggunakan kernel Linux merupakan salah satu distro Linux yang populer dengan kestabilannya. Dengan memperhitungkan distro berbasis Debian, seperti Ubuntu, Xubuntu, Knoppix, Mint, dan sebagainya, maka Debian merupakan distro Linux yang paling banyak digunakan di dunia.

Debian 5 adalah system operasi linux yang dapat dijadikan sebagai server. Dan yang akan di konfigurasi, DHCP Server, DNS Server, Web Server, Proxy Server. Sebelumnya kita harus mengkonfigurasi IP Address server terlebih dahulu.

1. Network Interfaces

  • Pertama, untuk mengatur IP Address tuliskan perintah, nano /etc/network/interfaces
  • lalu tulis address, netmask, network dan broadcast, seperti pada gambar berikut , lalu tekan CTRL + X lalu Y untuk menyimpan konfigurasi.
  • Setelah itu restart interface nya dengan perintah, /etc/init.d/networking restart
2. DHCP Server
  • Instal DHCP Server dengan perintah, apt-get install dhcp3-server
  • Lalu, edit DHCP dengan perintah, nano /etc/dhcp3/dhcpd.conf
  • lalu hapus tanda pagar yang ada pada tulisan subnet sampai max-lease-time, ubah subnet, netmask, range, option route, option broadcast-address, seperti pada gambar berikut:
  • Setelah itu, restart DHCP Server dengan perintah, /etc/init.d/dhcp3-server restart
  • Setelah konfigurasi DHCP Server selesai. Untuk melihat berhasil atau tidaknya DHCP Server cek pada komputer client apakah IP Address client sudah sama dengan IP Address yang telah di setting pada komputer server.
3. DNS Server
  • Install DNS nya dengan perintah, apt-get install bin9
  • Lalu edit file named.config dengan perintah, nano /etc/bind/named.conf
  • Ganti tulisan localhost menjadi nama domain seperti hasbi.net, db.local menjadi db.hasbi, dan 127 menjadi 192, lalu simpan.
  • Lalu copy file db.local ke db.hasbi
  • Lalu copy file db.127 ke db.192
  • Edit file db.hasbi, dengan perintah, nano /var/cache/bind/db.hasbi
  • Lalu ganti tulisan localhost menjadi hasbi.net
  • Edit file db.192 dengan perintah, nano /var/cache/bind/db.192
  • Lalu edit file resolv, dengan perintah nano /etc/resolv.conf
  • Lalu tulis, search hasbi.net dan nameserver 192.168.1.1
  • Lalu restart bind9 nya, dengan perintah /etc/init.d/bind9 restart
  • Lalu cek dengan perintah nslookup 192.168.1.1
  • Lalu cek dengan perintah nslookup hasbi.net
4. Web Server
  • Pertama install apache2 nya, dengan perintah apt-get install apache2
  • Edit apache2 nya, dengan perintah nano /etc/apache2/apache2.conf
  • Setelah itu, akan muncul tampilan berikut, tambahkan ServerName hasbi.net, lalu simpan.
  • Lalu edit site dengan perintah nano /etc/apache2/sites-available/default
  • Akan muncul tampilan seperti ini, lalu tambahkan admin@hasbi.net
  • Lalu edit tampilan website nya, dengan perintah nano /var/www/index.html
  • Akan muncul tampilan berikut, edit sesuai dengan keinginan.
  • Setelah itu restart apache2 nya, dengan perintah /etc/init.d/apache2 restart
  • Lalu cek di client apakah berhasil atau tidak, dengan menulis di browser nama domain hasbi.net atau IP Address server 192.168.1.1
5. Proxy Server
  • Install squid, dengan perintah apt-get install squid
  • Lalu edit squid dengan perintah, nano /etc/squid/squid.conf
  • lalu cari kata-kata, dengan menekan CTRL + W , cache_mem, lalu ganti menjadi cache_mem 64 MB
  • lalu cari kata-kata, cache_mgr tambahkan email contoh, hasbi99@gmail.com
  • cari kata-kata visible_hostname tambahkan hasbi.proxy.info.net
  • cari http_port 3128, lalu tambahkan transparent.
  • Cari kata-kata acl CONNECT
  • Cari http_access deny all, lalu tambahkan tanda pagar
  • Setelah semuanya di edit atau di tambahkan simpan dengan menekan CTRL+X lalu tekan Y.
  • Setelah itu tambahkan url atau link yang akan di block, dengan perintah nano /etc/squid/url
  • Akan muncul tampilan berikut, lalu tambahkan url atau link yang akan di block
  • Setelah itu tambahkan key atau kata kunci atau link yang akan di block, dengan perintah nano /etc/squid/key
  • Akan muncul tampilan berikut, lalu tambahkan key atau kata-kata yang akan di block
  • Setelah semua konfigurasi selesai restari squid nya, dengan perintah /etc/init.d/squid restart
  • Setelah itu cek di client apakah berhasil atau tidak dengan cara buka browser contohnya Mozilla firefox, atur dulu proxy pada browser nya klik option – advanced –network – setting, lalu pilih Manual proxy configuration, lalu isikan IP Address server 192.168.1.1 dan isi port 3128, klik OK
